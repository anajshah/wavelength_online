# Wavelength

## Overview

I wanted to be able to play wavelength without buying it (so expensive and for what reason...). 

## Tech Stack

I used Godot as my game engine, gd.script to program, procreate for drawings, and canva/google slides for the shapes.

## How It Works

The spinner is basically three area2ds layered on top of each other. The one on the bottom is the wheel part with the colors that rotates a random amount of time. The second one is the cover which rotates 180 and -180 based on its current position. The thrid one is a red spinner that you can move by hovering over it/the area around and drag it to place.
